Memory adalah perangkat atau sistem yang digunakan untuk menyimpan informasi untuk penggunaan langsung dalam komputer atau perangkat keras komputer dan perangkat elektronik digital yang terkait.

Cara Kerja Memory

Memori pada komputer bekerja sebagai penyimpan informasi yang diperlukan untuk menjalankan program dan menyimpan data. Ada beberapa jenis memori dalam sebuah komputer, dan mereka bekerja bersama-sama untuk memastikan operasi yang lancar. Secara umum, saat komputer dihidupkan, program dan data dimuat dari penyimpanan permanen (HDD atau SSD) ke RAM. CPU kemudian bekerja dengan data dalam RAM untuk menjalankan program, dan hasilnya mungkin disimpan kembali ke penyimpanan permanen. Proses ini memastikan kecepatan akses tinggi untuk CPU dan efisiensi dalam menjalankan tugas-tugas komputasi.
